## Key Features

1. **회원가입 및 로그인**
    - 서비스 자체 회원가입 및 로그인 지원
2. **온보딩**
    - 회원가입 후 온보딩 화면을 통해 서비스의 이념과 가치를 소개
3. **홈**
    - 기르고 있는 식물의 물주기 현황 확인 및 물주기
    - 친구와의 대화 목표 진행 상황 확인
4. **채팅**
    - 친구와 실시간으로 채팅 가능
    - AI 모드를 통해 대화 주제에 맞는 텍스트를 추천받아 전송 가능
5. **정원**
   - 친구와 번갈아 물을 주며 함께 식물을 키울 수 있음
   - 다 자란 식물은 정원에 심어 컬렉션으로 모을 수 있음
   - 공유일기 작성 및 하루한컷 미션 참여를 통해 영양제를 획득하고, 시든 식물에 사용 가능
6. **하루기록**
    - 공유일기: 감정과 이미지를 함께 담아 일기를 작성하고 친구와 공유 가능. 댓글 및 좋아요 기능을 통해 커뮤니티 활동 지원
    - 하루한컷: 매일 바뀌는 미션 주제에 맞는 사진을 올려 친구들과 일상 공유
7. **친구 초대**
    - QR 코드 및 개인 코드를 통해 친구 추가 가능
8. **친구 설정**
    - 친구 추가 후 또는 친구 설정 페이지에서 닉네임, 대화 빈도, 말투(존댓말/반말), 대화 주제 설정 가능
9. **마이페이지**
    - 프로필 사진 변경, 알림 설정, 이용약관 확인 가능
